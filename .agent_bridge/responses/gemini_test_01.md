이 프로젝트는 `airwriting_imu_only`라는 이름에서 알 수 있듯이, IMU(관성 측정 장치) 센서를 이용한 '허공 글씨 인식'(Air Writing) 시스템으로 보입니다.

주요 구성 요소는 다음과 같습니다:

1.  **펌웨어 (`firmware/`):** ESP32 마이크로컨트롤러에서 실행되는 코드로, IMU 센서 데이터를 수집하여 전송하는 역할을 합니다.
2.  **데이터 처리 및 머신러닝 (`imu/`, `fusion/`, `ml/`):** Python 기반의 백엔드입니다. ESP32로부터 데이터를 수신(`imu/`)하고, 센서 퓨전(`fusion/`)을 통해 움직임 궤적을 계산한 뒤, 머신러닝 모델(`ml/`, `models/`)을 사용하여 문자를 인식합니다.
3.  **데이터셋 (`data/`):** 문자 인식을 위한 EMNIST 및 IAM 필기 데이터셋을 포함하고 있습니다.
4.  **애플리케이션 (`web/`, `android_app/`):** 인식 결과를 보여주기 위한 웹 프론트엔드와 안드로이드 앱이 포함되어 있습니다. `ws_server.py`는 웹소켓 서버로 백엔드와 웹 프론트엔드 간의 실시간 통신을 담당하는 것으로 보입니다.

요약하자면, **임베디드 하드웨어(ESP32)에서 센서 데이터를 수집하고, 이를 Python 백엔드에서 처리하여 머신러닝으로 문자를 인식한 뒤, 웹 및 모바일 앱으로 결과를 보여주는 복합적인 시스템**입니다.
