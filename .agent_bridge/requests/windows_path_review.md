## 🔍 [AIOps 오케스트레이터] Windows 환경 경로 및 오류 검증 요청

**@Claude** 
현재 `airwriting_imu_only` 프로젝트를 새로운 Windows PC 환경(`C:\Users\USER\airwriting_imu_only`)으로 가져왔습니다. 
이 코드가 현재 윈도우 환경에서 실행될 때 발생할 수 있는 경로 문제나 포트 에러 등을 전체적으로 검증해 주시기 바랍니다.

### 주요 검토 포인트
1. **시리얼 포트 하드코딩 (`COM3`)**: `main.py`와 `imu/serial_receiver.py` 등 여러 곳에 `COM3`가 하드코딩 되어 있거나 우선순위로 잡혀 있습니다. 현재 Windows 시스템의 실제 포트와 충돌하거나, 포트 감지 로직이 실패할 가능성이 없는지 확인해 주세요.
2. **파일 경로 구분자 (Slash vs Backslash)**: `os.path.join` 대신 `/`나 `\\`를 섞어 쓴 곳이 있는지(예: 데이터셋 로더, 모델 가중치 로드 등) 검토해 주세요.
3. **권한 및 가상 환경**: Windows 특성상 발생할 수 있는 소켓(Socket) 권한이나 포트(`12347`, `8000`) 충돌 가능성을 점검해 주세요.

### 요청 사항
전체 코드(특히 `main.py`와 `imu/` 폴더 내의 직렬 통신 관련 코드)의 구조적 취약점을 분석하고, **Windows 환경에서 오류 없이 돌아갈 수 있도록 개선된 코드 스니펫이나 조언**을 작성해 주세요.
